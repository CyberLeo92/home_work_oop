# Интернет-магазин (E-commerce)

## Описание:
Проект представляет собой базовую реализацию ядра интернет-магазина с использованием ООП-подхода.
Подготовим всё для того, чтобы у нас появилось ядро для интернет-магазина. 
В дальнейшем для этого ядра возможно будет реализовать любой интерфейс — от сайта до телеграм-бота.

### Работа проекта:
#### _Функции_:
Функция в модуле `product.py`
1. **Основные сущности**:
   - Класс `Product` - товар с характеристиками:
     - Название (`name`)
     - Описание (`description`)
     - Цена (`price`)
     - Количество на складе (`quantity`)
   - Класс наследник `Smartphone` класса `Product` расширен характеристиками:
     - производительность (`efficiency`)
     - модель (`model`)
     - объем встроенной памяти (`memory`)
     - цвет (`color`)
   - Класс наследник `LawnGrass` класса `Product` расширен характеристиками:
     - страна-производитель (`county`)
     - срок прорастания (`germination_period`)
     - цвет (`color`)
   
2. **Дополнительные возможности**:
   - Сложение определённой категории товаров и получение общей суммы

3. Пример использование:
4. 
**Создание смартфона**:

phone = Smartphone("iPhone 15", "512GB", 100000, 10, 95.5, "15 Pro", 512, "Black")

**Создание газонной травы**:

grass = LawnGrass("Трава", "Зелёная", 500, 100, "Россия", "14 дней", "Зелёный")

**Сложение товаров одного типа**:

total = phone1 + phone2  # Корректно
total = grass1 + grass2  # Корректно
total = phone + grass    # Вызовет TypeError

Функция в модуле `category.py`
1. **Основные сущности**:
   - Класс `Category` - категория товаров:
     - Название (`name`)
     - Описание (`description`)
     - Список товаров (`products`)

2. **Дополнительные возможности**:
   - Автоматический подсчет количества категорий (`category_count`)
   - Автоматический подсчет общего количества товаров (`product_count`)


Функция в модуле `services.py`
Функция `load_categories_from_json`Загружает данные о категориях и товарах из JSON-файла 
и создает соответствующие объекты классов с проверкой дубликатов товаров


### Тестирование:
Тестирования происходит всех вышеописанных функций, для этого перейдите в папку "test" 
и в них запускать по отдельности либо предварительно установить через Poetry
```
poetry add --group dev pytest
```
И запустить `pytest` через терминал
